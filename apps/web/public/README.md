# Public assets

PWA icons and Open Graph image are currently placeholders copied from `src/media/radikal-logo.png`.

For production, regenerate square versions:

- `icon-192.png` — 192x192 PNG (maskable safe zone)
- `icon-512.png` — 512x512 PNG (maskable safe zone)
- `apple-touch-icon.png` — 180x180 PNG
- `og-image.png` — 1200x630 PNG (Open Graph / Twitter card)

Use tools like [maskable.app](https://maskable.app/editor) for maskable icons and [realfavicongenerator.net](https://realfavicongenerator.net/) for a full icon set.
